This project is a small Twitter-like app using Python with Neo4J driver to make Cypher queries. 

# Setup

1. Install [Python](https://www.python.org/downloads/)
2. Install [neo4j](https://neo4j.com/download/)
3. In neo4j desktop, create an instance and, if wanted, a database inside that instance (default is neo4j)
4. Start the instance
5. Change `credentials.json` so the data is aligned with the configurations of your neo4j
6. Install all dependencies of the project using `pip install -r requirements.txt` or `pip install neo4j` in a terminal instance in the project folder
7. Run `python loader.py` in the terminal to populate the database for your tests
8. Run the interface code to have full-access to Tweetyer using `python interface.py`!
